# TaxaCancelamentoClientes
Projeto realizado no curso Jornada Python da Hashtag Treinamentos utilizando python que detecta os principais motivos de cancelamento dos serviços de uma empresa, e soluções para esse problema.
Realizo importação dos dados CSV, visualização, tratamento de dados, exibição de gráficos e interpretação destes, e por fim, a sugestão de uma solução.
